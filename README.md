# Meta Ad Performance Dashboard -Powerbi

An interactive Power BI dashboard designed to analze Meta (Facebook & Instagram) advertising performance across engagement, conversion, demographic, and geographic dimensions.

---

## Project Overview

This project provides a comprehensive view of digital marketing performance by visualizing:

- Impressions and Clicks
- Engagement Metrics
- Conversion Performance
- Audience Demographics
- Geographic Distribution
- Campaign & Ad Type Analysis

The dashboard helps marketing teams optimize budget allocation, improve targeting strategies, and enhance ROI through data-driven insights.

---

## Key Performance Indicators

- Impressions: 216K+
- Clicks: 25K+
- Shares: 1.3K+
- Comments: 2.6K+
- Purchases (Conversions): 1.3K+
- CTR (Click-Through Rate): ~11.7%
- Engagement Rate: ~13.5%
- Conversion Rate: ~5.2%
- Purchase Rate: ~0.6%
- Total Budget: 2.5M

---

## Dashboard Features

### Performance Metrics
- KPI Cards for high-level performance overview
- CTR, Engagement Rate, Conversion Rate tracking
- Budget utilization insights

### Audience Insights
- Engagement by Gender
- Engagement by Target Age
- Engagement by Country (Interactive Map)
- Engagement by Calendar Month

### Trend Analysis
- Weekly Engagement Trends
- Hourly Engagement Trends

### Ad Performance Analysis
- Breakdown by Ad Type (Carousel, Image, Stories, Video)
- Performance comparison across CTR, Conversion Rate, and Engagement Rate

### Interactive Filters
- Select Measure (Engagements, Clicks, etc.)
- Campaign Name
- Target Interests
- Platform Selection (Facebook / Instagram)
